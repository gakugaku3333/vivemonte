# 検証計画: Phase 2b系統差(−1.71%)の新仮説「コンプトンエネルギー移行分布の差」

作成日: 2026-07-17。[PDD_RESULTS.md](PDD_RESULTS.md)で未解明のまま残った
kerma絶対値スケールの系統差（EGS5がChatCarloより全47ビン一貫して約−1.71%低い）の
原因究明計画。実行前に判定基準を事前登録する（本プロジェクトの相互検証運用に準拠）。

## 背景: 光電データソース差仮説は棄却済み（2026-07-17）

PDD_RESULTS.mdの有力仮説だった「PEGS5とNIST XAAMDIの光電吸収断面積の食い違い」を、
[extract_pegs5_photo_xs.py](extract_pegs5_photo_xs.py)（Phase 2aの全断面積照合と
同じPWLF再構成手法、AIRDET全断面積0.18628 cm²/gを相対−0.0016%で再現し手法を自己検証済み）
でPEGS5内部データから直接照合した結果:

| 成分 | PEGS5抽出値 [cm²/g] | xraylib(EPDL) [cm²/g] | 相対差 |
|---|---|---|---|
| 光電吸収 | 0.014987 | 0.014926 | **+0.408%** |
| コンプトン | 0.176906 | 0.177029 | −0.070% |
| レイリー | 0.013908 | 0.013918 | −0.070% |
| 全断面積 | 0.205802 | 0.205873 | −0.035% |

光電はkermaの46.8%を占めるが、+0.408%の差はkerma全体に**+0.19%**しか効かず、
しかも**符号が逆**（EGS5を高くする方向。観測はEGS5が低い）。同符号で一部を説明する
「部分的要因」の条件すら満たさないため、光電データソース差仮説は棄却。
残差はむしろ約−1.9%に広がった。

## 新仮説の定式化

カーマ近似（高ECUT）でのEGS5の局所エネルギー付与は、単位飛程あたり

    kerma_EGS5 ∝ σ_photo(PEGS5)×E + Σ_compton(PEGS5)×⟨T⟩_sampling

（レイリーは付与ゼロ。60 keV水では蛍光X線はO K殻0.543 keV・蛍光収率0.83%で
Augerがほぼ全て局所回収されるため光電は全エネルギー付与とみなせる）。
一方ChatCarloは

    kerma_ChatCarlo ∝ E×μen/ρ(NIST XAAMDI) = E×0.03190 cm²/g（60 keV水）

μen/ρの内訳は光電成分≈0.014926、コンプトン移行成分≈0.016974 cm²/g（53.2%）。
XAAMDI暗黙の平均移行率 t_XAAMDI = 0.016974/0.177029 ≈ **0.0959**。

**確定済みの事実（2026-07-17確認）**: Phase 1/2bの全EGS5ランはPEGS5入力
`IRAYL=1, IBOUND=1, INCOH=0, ICPROF=0, IMPACT=0`（`run_pdd60_phantom/pgs5job.pegs5inp`
3行目等）。IBOUND=1は**全断面積のみ**束縛補正し、INCOH=0のため散乱光子の
エネルギー・角度抽選は**自由電子Klein-Nishinaのまま**（S(q)棄却なし）、
ICPROF=0でDoppler広がりもなし。

**仮説**: 自由電子KNの平均エネルギー移行率 t_KN は、XAAMDI評価（S(q)補正込みの
散乱理論に基づく）の暗黙値 t_XAAMDI より小さい。S(q)は前方散乱（低エネルギー移行）を
抑制するため、S(q)補正込みの評価では平均移行率が上がる——EGS5(INCOH=0)はこの
持ち上げを受けないので、コンプトン成分のkermaが系統的に低くなる。

観測の−1.71%を再現するのに必要な値: 0.176906×t = 0.03190×(1−0.0171)−0.014987
→ **t_required ≈ 0.0925**、すなわち t_XAAMDI比で**約−3.5%**。この規模のS(q)効果が
60 keVで存在するかが検証対象。

## 事前登録の判定基準

Step 0の机上検算で予測される系統差 Δ_pred = (kerma_EGS5_eff − kerma_ChatCarlo)/kerma_ChatCarlo
（kerma_EGS5_eff = σ_photo(PEGS5) + Σ_compton(PEGS5)×t_KN、60 keV一次光子成分）
を、観測値と照合する:

- **主因と判定**: Δ_pred が −1.2%〜−2.2%（観測の平均−1.71%±0.5pp。ビン別観測は
  −0.71〜−3.45%で散乱スペクトルの混合により変動するため、平均に対する幅で判定）
- **棄却**: |Δ_pred| < 0.5%（光電仮説と同様、説明力なしとして棄却）
- **部分的要因**: 上記の中間（−0.5%〜−1.2%）。この場合は残差の追加要因を
  「既知の系統差・未確認事項」に戻す
- 符号が正なら大きさによらず棄却

Step 2（計装再実行）まで進んだ場合は、EGS5実測の⟨T⟩がStep 0のt_KN×60 keVと
統計誤差2σ以内で一致することを「EGS5がKN移行分布で動いている」ことの確認基準とする。

## Step 0: 机上検算（メインセッション、EGS5実行不要）

純Python+xraylibのデスクチェック（`check_compton_transfer.py`を新設）:

1. **自由電子KNの平均エネルギー移行率 t_KN(60 keV)**: エネルギー移行断面積
   σ_tr,KN の解析式（またはKN微分断面積×T の数値積分）÷ KN全断面積。
   実装は既存 `check_kn_vs_epdl.py` の解析積分と同系（照合可能）。
2. **S(q)重み付き平均移行率 t_Sq(60 keV)**: KN微分断面積×S(q)（xraylib
   `SF_Compt`、H/Oを質量分率合成）×T の数値積分 ÷ S(q)重み付き全断面積。
   これがXAAMDI側の物理に対応する近似値。
3. **XAAMDI暗黙値との三者比較**: t_KN / t_Sq / t_XAAMDI(=0.0959) を並べ、
   t_Sq ≈ t_XAAMDI（XAAMDIの物理を再現できているか）と t_KN の乖離幅を確認。
4. **Δ_pred の算出と判定基準への照合**（上記事前登録基準）。
5. **エネルギー依存の副検算**: 同じ計算を20/30/40/50 keVでも行い、Δ_pred(E)の
   形を出す。深いビンほど散乱で軟化したスペクトルが混ざるため、観測のビン別
   ばらつき（−0.71〜−3.45%）とΔ_pred(E)の傾向が矛盾しないかを定性確認する
   （厳密なスペクトル畳み込みはStep 2以降のオプション）。

成果物: スクリプト＋実行ログ＋本計画書への結果追記。

### Step 1 実行結果（2026-07-17、egs5-operator）

`egs5_compt.f`のコンプトン抽選ループ（サブルーチンCOMPT、54–98行）を直接読み、
Step 0の前提を実装レベルで確認:

- 棄却関数`rejf3 = 1. - br*sinthe/(1. + br*br)`（80行）は自由電子KNそのもの。
  S(q)補正ブロック（83–97行）は`incohr(irloc)==1`のときのみ実行され、
  `rejf3`に散乱関数`sxz`を追加で掛ける構造。
- **実行時のフラグ値まで確認**: `egs5_block_set.f`のデフォルト初期化
  （`incohr(i)=0`、全リージョン）を`pdd60_phantom.f`が一度も上書きしていない
  （`grep incoh`が0件）。したがってPhase 2bランでは実際に`incohr(2)=0`のまま
  実行され、S(q)棄却ブロックは常にスキップ——「INCOH=0だから自由電子KNのはず」
  という推測ではなく、実装・実行時設定の両方から裏付けが取れた。
- IBOUNDは`comptm`関数（PEGS5、全断面積計算）にのみ効き、COMPTサブルーチン
  245行中に`ibound`という識別子は一度も出現しない——角度・エネルギー抽選には
  無関係であることをソース上で確認。
- 対照確認: INCOH=1はS(q)重み付き棄却のみ追加（esgの計算式自体は不変）、
  ICPROF（Doppler broadening）はINCOH=1必須の上位追加補正（`hatch.f`
  478–489行で強制チェック）——今回のランはどちらも入っていない。

**未確認のまま残る点**: GMFP/GBR1/GBR2を実際に消費するEGS5の`photon`ルーチン
（飛程・相互作用種別選択部）自体は未読——IBOUNDがそこで別の形で再利用されて
いないかは見ていない。ただしCOMPT内のサンプリング過程への影響については
確定的な結論。

### 総合判定（Step 0+1、2026-07-17時点）

「コンプトンエネルギー移行分布の差（IBOUND=1×INCOH=0の混成構成）」は
**部分的要因として実装レベルで裏付け済み**（観測−1.71%の約64%を同符号・
同エネルギー依存傾向で説明）。残り約36%（−0.6pt程度）は未解明のまま
——Step 2（EGS5計装再実行）またはStep 3（文書化・残差の別要因調査）に進む
判断はユーザーに委ねる。

## 落とし穴・限界（事前に認識しておくもの）

- **μtr vs μen**: 60 keV水では放射損失は無視できる（差<0.1%）ためμtr≈μenと
  して扱う。判定基準の幅0.5ppに対して十分小さい。
- **XAAMDIの内訳の近似**: t_XAAMDI はμen/ρからXCOM系光電成分を引き算した
  暗黙値であり、Hubbell-Seltzerの評価内訳を直接見たものではない。Step 0の
  t_Sq計算がこの近似の妥当性チェックを兼ねる（t_Sq≈t_XAAMDIなら整合）。
- **一次光子近似**: Step 0のΔ_predは60 keV一次成分の値。実測ビンは後方散乱・
  多重散乱の軟化スペクトルを含む（表面ビンでもBSF≈1.48相当の散乱寄与）。
  平均−1.71%との照合には±0.5ppの幅を持たせて対応（事前登録基準に織り込み済み）。
- **PEGS5のPWLF分解能**: 抽出値はbin=23の区分線形フィット値であり、真のテーブル値
  と微小差がありうる（AIRDET再現で−0.0016%と確認済み、無視できる）。
- 仮説が主因と確定した場合の帰結は「どちらかのバグ」ではなく**物理モデルの構成差**
  （Phase 1のIBOUND=0/1の教訓と同型）。対処は修正ではなく、比較条件の明記
  （EGS5側をINCOH=1にして再実行するか、ChatCarlo側の推定量の定義を明記するか）
  になる見込み。

## Step 2 実行結果（2026-07-17、egs5-operator、`docs/egs5_crosscheck/pdd60_phantom_v2/`）

既存の47ビンPDD/OCRタリー（`pdd60_phantom.f`）はロジック変更なしでそのまま残し、
ausgabにコンプトンエネルギー移行タリーと光電/コンプトン別edep内訳を追加した
`pdd60_phantom_v2.f`をn=10,000,000で再実行（新規サブディレクトリ、既存ファイルは
変更なし）。「一次」判定はPhase 1と同じLATCH方式（コンプトン散乱を一度も受けていない
光子＝latch(np)==0、tutor5と同一の慣習）で、これにより「一次」タリーの光子エネルギーは
必ず厳密に60 keVであることが保証される（Step 0のt_KN(60 keV)と直接比較可能）。

### 結果

| 区分 | N events | ⟨T⟩ [keV] | 統計誤差(1σ) |
|---|---|---|---|
| **一次**（60 keV厳密、latch=0時点） | 9,027,250 | 5.617222 | ±0.001199（相対0.0213%） |
| 全コンプトン（多重散乱込み） | 30,233,185 | 4.359490 | ±0.000571（相対0.0131%） |

```
t_measured(一次) = <T>/60keV = 0.093620 ± 0.000020
Step 0 desk-check t_KN(60 keV)  = 0.093630
差 = -0.000010  →  -0.482σ
```

**2σ以内で一致（実際は0.48σ）**——事前登録した判定基準を明確にクリア。Step 0/1が
理論・コード読解ベースで示した「EGS5は自由電子Klein-Nishinaでコンプトン散乱を
サンプリングしている」という結論が、実際のMC出力（60 keV厳密の一次コンプトン
イベント900万件超の直接集計）でも独立に再現された。

チャンネル別edep内訳（ファントム全域、多重散乱込みの全エネルギー）:
光電由来242174.27 MeV・コンプトン由来131801.27 MeV・合計との整合（回帰チェック）も
桁まで一致（sumtot=373975.54 MeVと合致）。光電が全edepの64.8%を占めるのは、
60 keV一次光子の相互作用確率としては光電がわずか7.8%しかない
（**定義: photo/(photo+compton)、xraylib断面積比、レイリーは弾性散乱でエネルギー
付与競合チャネルに含まれないため分母から除外** — vive-auditor所見6で分母定義の
明記を指摘され追記。レイリーを含めた全断面積比だと7.25%になり0.5pt異なるが、
以下の定性的結論は変わらない）ことと矛盾するように見えるが、多重散乱で軟化した
光子ほど光電吸収確率が急増する（Step 0のエネルギー依存性表: 20 keVで75.4%、下記
「副検算」節参照）ため、edep（吸収エネルギー）ベースでは低エネルギー2次光子の
寄与が支配的になる——物理的に整合的で異常ではない。

回帰チェック: 既存47ビンタリーの値もv2で再現されることを確認（例: z0-1ビン、
v1(n=1e8)の0.116829 keV/history に対しv2(n=1e7)は0.116087 keV/history、差−0.64%は
v2の統計誤差0.5442%(1σ)の1.17倍——別seed・別nでの再実行として想定内の統計揺らぎ。
Step 2の新規タリー追加が既存ロジックを壊していないことを確認）。

### Step 0+1+2 総合結論

「コンプトンエネルギー移行分布の差（IBOUND=1×INCOH=0でEGS5が自由電子KNのまま
散乱している）」は、机上検算（Step 0）・ソースコード読解（Step 1）・実際のMC実行に
よる直接測定（Step 2、0.48σ一致）の**三重に独立な手法で確認された部分的要因**。
観測された−1.71%のうち約64%（−1.095%）を同符号・同エネルギー依存傾向で説明する。
残り約36%（−0.6pt程度）は依然未解明のまま。
