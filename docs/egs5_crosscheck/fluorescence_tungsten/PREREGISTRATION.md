# 事前登録: K殻蛍光X線実装のEGS5相互検証（タングステン、W, Z=74）

作成日: 2026-07-18（この時刻のChatCarlo/EGS5双方の実行結果は未取得。以下は
結果を見る前に固定する）

## 検証したい実装

[docs/plan_fluorescence.md](../../plan_fluorescence.md)で実装したK殻蛍光X線
（コミット b732d35 / a6f133c、8線化修正 88ee64b）。鉛での検証
（[docs/egs5_crosscheck/fluorescence/RESULTS.md](../fluorescence/RESULTS.md)、
条件付き合格）・銅での検証（[docs/egs5_crosscheck/fluorescence_copper/RESULTS.md](../fluorescence_copper/RESULTS.md)）
に続く第3弾。タングステンはX線管の陽極（ターゲット）材料として実運用で
使われる（auto memory `project_status.md`の次セッション積み残し項目）。
方法論は鉛・銅検証と同一（100 keV単色鉛筆ビーム→金属スラブ透過）。

## 幾何・線質

- 単色100 keV光子の鉛筆ビームがタングステン(W)スラブに垂直入射
- スラブ厚さ: **0.035 cm**。理由: `linear_mu(W, 100keV) ≈ 85.63 /cm`
  （1/e深さ≈0.0117cm）なので、この厚さは約3.00平均自由行程分
  （鉛0.05cm≈3.15mfp、銅0.75cm≈3.08mfpと同じ考え方でmfp数を揃えて選定）
- history数: 両コードとも1,000,000
- ChatCarlo seed=1、EGS5 inseed=1
- 物理モデル: IBOUND=1・IRAYL=1・ICPROF=0を両コードで揃える
- 蛍光フラグはON/OFFの2条件

## 事前に予測しておく物理的傾向（結果を見る前に明記）

タングステンは鉛(Z=82)に近い高Z元素であり、K吸収端(69.525 keV)・Kライン
群（Kα2 57.981/Kα1 59.318/Kβ1 67.244/Kβ2 69.100 keV、xraylibで確認済み）の
エネルギーオーダーは鉛（K吸収端88 keV、Kライン72-85 keV）と近い。銅で
見られた「蛍光線が桁違いに低エネルギーで自己吸収が支配的」という非対称性は
タングステンでは想定していない:

- `linear_mu(W, 100keV) ≈ 85.63 /cm`に対し、`linear_mu(W, 58-69keV)`は
  50.0–78.2 /cm と、100 keVでのmuと**同程度〜やや小さい**（鉛の場合ほど
  劇的にmuが下がる訳ではないが、上がりもしない）。したがって鉛と同程度に
  蛍光光子がスラブを脱出できると予測する。
- xraylibで確認済みの光電吸収パラメータ: W@100keVで光電分率93.6%・K殻
  分率81.6%・ω_K=0.954（鉛・銅よりさらに高い）。蛍光生成レート自体は
  検証対象の3元素中もっとも高いと予測する。
- **予測**: 観測量B（蛍光ピーク帯脱出割合）はON条件でOFF条件の2倍以上に
  増加する、という鉛検証と同種の傾向が再現されるはずである
  （銅のような「非情報的になる」例外扱いは不要と予測）。
- **予測**: 観測量Aの差分Δ(ON-OFF)は、鉛検証のΔと同程度かそれ以上の
  大きさになると予測する（K殻蛍光生成レートが3元素中最大のため）。

これらの予測は結果と照合する対象であり、外れた場合はその原因を
「想定される既知の系統差要因」またはRESULTS.mdの追加調査項目として扱う。

## 評価する量と許容基準（結果を見る前に固定）

### 観測量A（主）: 脱出光子の全エネルギー透過率

`Σ(脱出光子のエネルギー) / (入射光子数 × 100 keV)`。鉛・銅検証と同一定義。

- **ステップ1（対照実験、蛍光OFF）**:
  **合格基準: 相対差2%以内かつ2σ以内**（鉛・銅検証と同一）。
- **ステップ2（蛍光ON）**:
  **合格基準: 相対差5%以内かつ3σ以内**（鉛・銅検証と同一。L殻蛍光の
  非対称を許容する幅。タングステンL吸収端は10.2-12.1 keV、L線は
  8.4-11.5 keV程度で、K線(58-69keV)よりだいぶ低エネルギー。鉛のL線
  10.5-12.6 keVとほぼ同オーダーであり、鉛検証時の系統差の議論がそのまま
  適用できると予測する）。
- **ステップ3（差分の一致、最も重視する）**: Δ = (観測量A)_ON - (観測量A)_OFF。
  **合格基準: ΔのChatCarlo/EGS5比が0.7〜1.3倍の範囲内**（鉛・銅検証と同一）。

### 観測量B（副・定性的）

帯域: タングステンのKライン群をカバーする56–70 keV
（Kα2 57.981/Kα1 59.318/Kβ1 67.244/Kβ2 69.100 keV相当）。

- **合格基準**: 蛍光ONでOFFの2倍以上に増える、という定性的傾向が
  **両コードで**再現されること（鉛検証と同一の判定方式。銅検証で導入した
  「非情報的になりうる」の例外扱いは、上記の予測に基づきタングステンには
  適用しない前提とする。ただし実行結果が予測に反してほぼゼロ同士の比較に
  なった場合は、銅と同じ理由で判定除外に切り替える — その場合も
  「予測が外れた」事実として記録する）。

## 想定される既知の系統差要因（結果を見る前に列挙）

- **L殻蛍光の非対称**: 鉛検証と同じ理由・同じ許容基準への織り込み。
- **線エネルギー値の出典差**: 鉛・銅検証と同じ。
- **K殻イオン化確率のモデル差**: 鉛・銅検証と同じ（観測量Aの許容基準5%に
  織り込み済み）。
- **密度**: タングステン密度はxraylib既定値（`resolve("W")`で確認済み:
  **19.3 g/cm³**）を使用する。銅・鉛と異なり`_DENSITY_OVERRIDE`に
  タングステンのエントリがない（`materials.py`参照）ため、EGS5 PEGS5入力の
  RHOをこの19.3 g/cm³に明示的に一致させることを依頼文で念押しする。

## 判定の扱い

- 上記基準に対して不合格が出ても基準そのものを緩めない。原因調査を別項目
  として記録し、未解決のまま「条件付き合格」として残すことを許容する
  （鉛検証と同じ運用）。
