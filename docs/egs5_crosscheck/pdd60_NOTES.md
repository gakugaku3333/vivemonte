# EGS5相互検証 Phase 2b（水ファントム深部線量分布PDD＋側方プロファイル、60 keV単色）

作成日: 2026-07-16。担当範囲はEGS5側のみ（ユーザーコード作成・ビルド・実行・出力取得・
ChatCarlo側結果との突合）。ChatCarlo側の実行スクリプトは
`docs/egs5_crosscheck/run_chatcarlo_pdd60.py`（既存、`chatcarlo_pdd60_results.json` に
n=5,000,000史の結果が保存済み）。

**結論を先に書く: この比較は事前登録した合否基準（`docs/plan_phits_crosscheck.md`
Phase 2の「統計誤差1%未満のビンに限定して2σかつ相対差2%以内」）に対して
不合格（47ビン中37ビンが不合格、6ビンのみ合格、4ビンは統計誤差条件外）。
EGS5がChatCarloより一貫して低く（全47ビンで同符号）、平均相対差−1.71%
（範囲−0.71%〜−3.45%）という系統的な差が見つかった。これは統計揺らぎでは
説明できない規模・符号一貫性であり、原因は未確定（詳細は「未確認事項」節）。
抄録には正直にこの結果を記載すべきで、「良好な一致」と書くのは不適切と判断する。**

## 幾何・線源条件（Phase 2aから完全流用）

- 60 keV単色光子、10×10 cm²照射野（SSD=100cm相当）、**非発散（平行ビーム）近似**
  （半開角2.9°を無視、光路長誤差0.13%程度 — Phase 2aと同じ採用理由）。
- 水ファントム: 30×30×20 cm、前面z=0。密度はPEGS5テンプレートのRHO=1.001 g/cm³
  （ChatCarlo側は1.000 g/cm³。既知の相対0.1〜0.2%系統差、Phase 1/2aから継続）。
- IBOUND=1固定（束縛コンプトン、Phase 1の教訓を踏襲）。IRAYL=1（レイリー散乱あり）。
- ecut=1.5（全エネルギー=運動エネルギー0.989 MeV相当、水60 keVのコンプトン電子
  最大運動エネルギー約11 keVを大きく上回るため、生成された二次電子は生成直後に
  カットオフ未満として即座に局所吸収される — カーマ近似が自動的に成立）。
  pcut=0.010 MeV。

## Phase 2aとの違い: ジオメトリの単純化とスコアリング方式

Phase 2aの`bsf60_phantom.f`は「前面層0-0.2cm」「バルク0.2-20cm」の2物理領域に
分けていたが、Phase 2bは深さビンが15段階に増えるため、物理領域を増やす代わりに
**単一の水領域（region 2、0≤z≤20、|x|≤15、|y|≤15）**とし、howfarを簡素化した
（RPP境界チェックからz方向の層分割ロジックを削除）。スコアリングは
`ausgab`内で相互作用点の解析座標(X,Y,Z)を47個のビン境界と直接比較する方式
（collision estimator、tutor2/Phase2a方式を踏襲）。

### ビン定義（ChatCarlo側`run_chatcarlo_pdd60.py`の`_build_bins()`と厳密一致）

- **PDD中心軸**（`pdd_z<i>-<i+1>`, 15ビン）: |x|≤1cm, |y|≤1cm, 深さz=0〜15cmを
  1cm刻み。各ビン体積4 cm³（2×2×1）。
- **側方プロファイル・表面近傍**（`lat_shallow_x<x0>-<x1>`, 16ビン）: z∈[0,1)cm,
  |y|≤1cm, xを−8〜8cmまで1cm刻み。各ビン体積2 cm³（1×2×1）。
- **側方プロファイル・10cm深**（`lat_10cm_x<x0>-<x1>`, 16ビン）: z∈[9,10)cm,
  |y|≤1cm, xを−8〜8cmまで1cm刻み。各ビン体積2 cm³。

合計47ビン。3グループは**グループ内では相互排他**（例: PDDの15ビンはzで単純分割、
1イベントは高々1ビンにヒット）だが、**グループ間では意図的に重なりを許す**
（例えば座標(0.5, 0.5, 0.5)の相互作用点は`pdd_z0-1`と`lat_shallow_x0-1`の
両方にヒットする — 独立集計であり、ChatCarlo側のtrack-length推定量も同じ設計
なので、これは整合性チェックに使える）。

**整合性チェック（自己検算）**: `pdd_z0-1` = `lat_shallow_x-1-0` + `lat_shallow_x0-1`
（同じz∈[0,1]・|x|≤1の領域を、PDDは1本の柱として、側方プロファイルは中心の
2本のx柱の和として集計しているはずなので、厳密に一致するはず）。
実測: 1.16829×10⁻⁴ MeV = 5.84418×10⁻⁵ + 5.83873×10⁻⁵ = 1.168291×10⁻⁴ MeV
（完全一致、丸め誤差のみ）。同様に `pdd_z9-10` = `lat_10cm_x-1-0` + `lat_10cm_x0-1`:
4.44520×10⁻⁵ = 2.22635×10⁻⁵ + 2.21884×10⁻⁵ = 4.44519×10⁻⁵（完全一致）。
**ausgabのビン分類ロジックにバグがないことをこの2点で確認した。**

- ユーザーコード: `pdd60_phantom.f`（`tutorcodes`ではなくPhase 2aの
  `bsf60_phantom.f`を土台に、howfarを単一領域に簡素化、ausgabに47ビン分類を追加）。
- PEGS5入力: `pdd60_phantom.inp`はPhase 1/2aと同一テンプレート（RHO=1.001,
  IRAYL=1, IBOUND=1, INCOH=0, ICPROF=0, IMPACT=0, AE=0.521, AP=0.0100,
  UE=0.580, UP=0.070）。
- inseed=1, luxlev=1（ranlux）。
- history数=100,000,000（1億）。事前に50,000史・1,000,000史・10,000,000史で
  ビルド・タイミングを検証し（このマシンでは10M史が約14秒と非常に高速だった
  ため）、指示にあった目安2-5M史より大幅に増やした。実測所要時間は約103秒
  （ビルド込み）。統計誤差はEGS5側は全47ビンで1%未満（最大0.89%、
  `lat_shallow_x±8-7`）を達成した。

## Gy/history換算とChatCarloとの比較

## 形状の物理的妥当性（別問題として）

- PDDは深さとともに単調減少（1.168×10⁻⁴→1.959×10⁻⁵ MeV/history、z=0〜15cm）
  しており、カーマ近似（電子飛程ゼロ）に予想される「ビルドアップ域のピークが
  出ない滑らかな減衰」という形状と整合する。EGS5・ChatCarlo双方でこの形状は
  一致している（形が違うのではなく、絶対値のスケールがずれている）。
- 側方プロファイルは中心軸(x=0)付近でほぼフラット（±4cm程度、照射野半幅
  5cmにほぼ対応）、視野端（|x|≈5cm）で急峻に立ち下がり、視野外（|x|>5cm）
  でも散乱由来のテール（半影＋体内散乱）が緩やかに続く — 定性的に期待される
  形状。EGS5・ChatCarloとも左右対称性が良い（中心軸±1cm内で最大差0.5%
  程度）。
- 以上より、**幾何・線源サンプリング・ビン分類のロジックは両コードとも
  正しく機能している**と判断する（形状の一致、上記の自己整合性チェック2件、
  Phase 2aの物理妥当性確認の継続）。問題は**絶対値スケール**に限定される。

## 未確認事項・既知の系統差

- **主要な未解決事項（今回新規発見）: −1.7%系統差の原因**。Phase 1・2aは
  いずれも「全断面積・透過率」（Compton束縛補正込みで相対0.88%、1.7σ一致）
  または「同一コード内の比」（BSF、両辺とも同じEGS5 collision estimator）を
  比較しており、**ChatCarloのμen/ρベースtrack-length kermaとEGS5の実物理
  シミュレーション局所吸収エネルギーを直接突き合わせたのはPhase 2bが初めて**。
  今回見つかった系統差はこの新しい比較軸で初めて露見した可能性が高い。
  有力仮説（未確証）: ChatCarloのkerma推定量は各セグメントで
  `mu_en_rho("water", E)`（NIST XAAMDI/Hubbell-Seltzer表、60 keVで
  0.03190 cm²/g、表の格子点そのまま使用）×セグメント長×光子エネルギーを
  積算する。一方EGS5のausgab `edep`はPEGS5が内部で生成した断面積データ
  （Storm-Israel系＋IBOUND束縛コンプトン補正）に基づく実際のモンテカルロ
  相互作用（光電吸収・コンプトン電子・レイリー無エネルギー損失）から
  積み上げた真の局所エネルギー付与であり、**NIST XAAMDIのμen/ρとPEGS5内部
  断面積データは独立に評価された別ソースのデータで、必ずしもサブパーセント
  精度で一致する保証がない**。全断面積（μ/ρ、透過率で検証済み）が良く一致
  していても、光電吸収の相互作用あたり平均エネルギー移行など「エネルギー
  付与の内訳」は別のデータセットに依存するため、こちらだけがずれている
  可能性がある。**この仮説は未検証**（PEGS5が実際に使っている光電断面積
  テーブルの値を取り出してNIST XAAMDIと直接比較する追加作業が必要、
  今回は時間の制約で未実施）。
- **密度差**: RHO=1.001 (EGS5) vs 1.000 (ChatCarlo) — 相対0.1%、上記−1.7%の
  主因にはなり得ない（1桁小さい）。
- **点源発散の近似**: 半開角2.9°を無視した非発散近似。両コード共通の近似
  なので、この比較の相対差には影響しないはずだが定量評価は未実施。
- **側方境界±15cmの効果**: ファントムを30×30cmの有限箱として実装。無限に
  広いファントムとの差は未評価（Phase 2aから継続の未確認事項）。
- **蛍光X線（特性X線）の非輸送**: 低Z（H, O）なので蛍光収率は無視できる
  水準（O原子のK殻蛍光収率約0.008）。−1.7%を説明する量ではないと考えるが、
  定量評価はしていない。
- **EGS5側の乱数シード変更による再現性確認**は今回も未実施（Phase 1/2aから
  持ち越しの既知の省略事項、時間の制約）。
- **中心軸PDD以外の妥当性確認**: 側方プロファイルの絶対値についても
  PDDと同様の−1.7%程度の系統差が見られ、系統差が特定の深さ・位置に
  局在するものではなく、ほぼ全ビンに一様にかかる「係数的」な差である
  ことを示唆する（相互作用点分布の形自体はほぼ一致しているため、幾何・
  輸送ロジックの誤りである可能性は低いと判断する根拠）。

## 抄録記載にあたっての推奨

この−1.7%系統差は**pre-registered基準に対する明確な不合格**であり、
「ChatCarloとEGS5は良好に一致した」という総括は現時点のデータでは書けない。
一方で、(1) 全断面積・一次透過率（Phase 1）は1.7σ・0.88%で一致、(2) BSF
（Phase 2a）はEGS5単独の自己無撞着な計算として妥当な値（1.453±0.024）を
示し、(3) 今回のPDD/プロファイルも**形状は良く一致し、原因は絶対値換算の
どこかにある**、という3点を切り分けて記述するのが正直な報告になる。
締切に間に合わせるための最短の書き方としては、「散乱を含む線量分布の
コード間比較で約1.7%の系統差を検出、原因究明中（cross-section
データソースの差が有力仮説）」という記載を提案する。「一致」と断定する
記述は避けること。
