# 事前登録: K殻蛍光X線実装のEGS5相互検証（銅、Cu, Z=29）

作成日: 2026-07-18（この時刻のChatCarlo/EGS5双方の実行結果は未取得。以下は
結果を見る前に固定する）

## 検証したい実装

[docs/plan_fluorescence.md](../../plan_fluorescence.md)で実装したK殻蛍光X線
（コミット b732d35 / a6f133c、その後の8線化修正 88ee64b）。鉛(Pb)での検証
（[docs/egs5_crosscheck/fluorescence/RESULTS.md](../fluorescence/RESULTS.md)、
条件付き合格）に続き、低Z・高蛍光収率側の代表として銅で追加検証する
（付加濾過材として実運用で使われる材料 — auto memory `project_status.md`の
次セッション積み残し項目）。方法論は鉛検証と同一（100 keV単色鉛筆ビーム→
金属スラブ透過）を踏襲する。

## 幾何・線質

- 単色100 keV光子の鉛筆ビームが銅(Cu)スラブに垂直入射（鉛検証と同一の
  LATCH方式・ジオメトリパターン、材料と厚さのみ変更）
- スラブ厚さ: **0.75 cm**。理由: `linear_mu(copper, 100keV) ≈ 4.108 /cm`
  （1/e深さ≈0.243cm）なので、この厚さは約3.08平均自由行程分
  （鉛検証の0.05cm≈3.15 mfpとほぼ同じmfp数で選定 — 絶対厚さでなくmfp数を
  揃えることで統計的な条件を鉛検証と揃える）
- history数: 両コードとも1,000,000
- ChatCarlo seed=1、EGS5 inseed=1
- 物理モデル: IBOUND=1・IRAYL=1・ICPROF=0を両コードで揃える（鉛検証と同一）。
  蛍光フラグはON/OFFの2条件

## 事前に予測しておく物理的な非対称性（結果を見る前に明記）

鉛検証との重要な違いが1つある。**この違いにより観測量B（蛍光ピーク帯の
脱出割合）が銅では非情報的になる可能性が高いことを、結果を見る前に
ここで予測しておく**:

- 銅のK吸収端は8.979 keV、Kα1/Kα2/Kβ1線energyは8.048/8.028/8.905 keV
  （xraylibで確認済み、`fluorescence_k_data(29)`と同じ値）。
- 鉛検証では蛍光線（72–85 keV）が「K吸収端(88 keV)より低いエネルギー」に
  あるため、K殻光電吸収が効かず`linear_mu(Pb, ~75-85keV)`は
  `linear_mu(Pb, 100keV)`よりむしろ**小さい**（23.7–34.8 /cm vs 63.0 /cm）。
  蛍光光子は比較的減弱を受けにくく、スラブから脱出しやすかった。
- 銅では蛍光線自体が絶対エネルギーとして極めて低い（8 keV前後）ため、
  K殻光電吸収が効かなくても光電効果自体がE^-3則で強く効き、
  `linear_mu(Cu, 8.0-8.5keV) ≈ 400-470 /cm`（1/e深さ≈21-25 μm）と、
  100 keVでのmu(4.1/cm)より**2桁大きい**。0.75cm厚のスラブでは、銅の
  Kα光子はスラブ内部で生成されるとほぼ確実に再吸収され、脱出面から
  最後の数十μm以内で生成されたものしか脱出できない
  （鉛の場合と逆方向の非対称性）。
- **予測**: 蛍光ON条件でも観測量B（72-86 keV相当、銅ではKライン帯
  7.8-9.2 keVに読み替え）の脱出割合は、OFF・ON双方でほぼゼロに近い値になり、
  「ONがOFFの2倍以上」という鉛検証の判定基準が数値的に意味をなさない
  （0近傍同士の比較になる）可能性が高い。この場合、観測量Bは「両コードで
  同程度にほぼゼロ」であることの定性確認にとどめ、**判定には使わない**
  （鉛の場合と扱いを変えることを、結果を見る前にここで宣言しておく）。

一方で、光電吸収イベント自体はxraylibの断面積計算で確認済みの通り、
銅は100 keVで光電分率62.8%・K殻分率89.6%・ω_K=0.454と、鉛より
K殻蛍光の**生成レート自体は高い**（鉛は光電が支配的でK殻ionization
自体も起きやすいが、銅は電子殻の相対的なK殻優位性が異なる）。したがって
観測量A（エネルギー収支）では蛍光生成自体の効果は十分な統計で捉えられる
はずであり、「生成されるが脱出しない（局所再吸収される）」という
描像が正しければ、**ON/OFFでの透過エネルギー割合の差Δは、鉛の場合より
小さくなる**と予測する（脱出せず結局その場で吸収されるなら、ONにしても
OFFの局所全吸収とエネルギー収支上ほぼ変わらないため）。この予測自体も
結果と照合する対象とする。

## 評価する量と許容基準（結果を見る前に固定）

### 観測量A（主）: 脱出光子の全エネルギー透過率

`Σ(脱出光子のエネルギー) / (入射光子数 × 100 keV)`。鉛検証と同一定義。

- **ステップ1（対照実験、蛍光OFF）**: 既存の相互検証範囲内（束縛コンプトン+
  光電+レイリーの銅内輸送、新材料での初検証）。
  **合格基準: 相対差2%以内かつ2σ以内**（鉛検証Step1・Phase 1水60keVと同じ）。
- **ステップ2（蛍光ON）**:
  **合格基準: 相対差5%以内かつ3σ以内**（鉛検証と同じ理由: ChatCarloはK殻の
  みでEGS5がL殻蛍光も含む可能性がある差を許容する幅。銅L吸収端は約0.93/0.95keV
  と非常に低く、L殻蛍光線もそれ以下のエネルギーになるため、鉛よりもさらに
  自己吸収されやすく実質的な寄与はほぼ皆無と予想されるが、確認前の推測に
  留め、基準自体は鉛と揃えて緩めない）。
- **ステップ3（差分の一致、最も重視する）**: Δ = (観測量A)_ON - (観測量A)_OFF。
  **合格基準: ΔのChatCarlo/EGS5比が0.7〜1.3倍の範囲内**（鉛検証と同一基準）。
  ただし上記の予測通りΔ自体が非常に小さい値になった場合、比の分母が
  ゼロに近づき比自体が不安定になりうる。その場合は絶対差
  `|Δ_ChatCarlo - Δ_EGS5|`が観測量Aステップ1の統計誤差の3倍以内であることを
  代替基準とする（これも結果を見る前にここで定義しておく — Δが小さいことが
  判明してから基準を都合よく変えたのではないことの担保）。

### 観測量B（副・定性的、銅では非情報的になる可能性を上記で予測済み）

帯域: 銅のKライン群をカバーする7.8–9.2 keV
（Kα2 8.028 / Kα1 8.048 / Kβ1 8.905 keV相当、鉛検証の72-86keVに対応する
銅版の帯域）。

- **合格基準**: 蛍光ONでOFFの2倍以上に増えることが**両コードで**再現される
  こと。ただし上記の予測通りON・OFFとも脱出光子中のこの帯域の割合が
  1%未満（ほぼゼロ）である場合、この基準は**判定に使わず**、「両コードで
  同程度に無視できる」ことの定性確認にとどめる（この扱いの変更は結果を見る
  前にここで宣言済み）。

## 想定される既知の系統差要因（結果を見る前に列挙）

- **L殻蛍光の非対称**: 鉛検証と同じ理由。銅のL吸収端(~1 keV)はさらに低く、
  L殻蛍光線も同程度に低エネルギーで、生成されても即座に局所吸収されるため、
  透過エネルギー収支への影響は鉛の場合よりさらに小さいと予想する。
- **線エネルギー値の出典差**: 鉛検証と同じ（xraylib vs EGS5内蔵テーブル）。
  銅は8 keV帯とエネルギーが低い分、出典差の相対的な影響が大きくなりうる。
- **K殻イオン化確率のモデル差**: 鉛検証と同じ理由（観測量Aの許容基準5%に
  織り込み済み）。
- **密度**: 銅密度はChatCarlo/xraylib基準で8.96 g/cm³固定
  （`materials.py: _DENSITY_OVERRIDE["Cu"] = 8.96`）。EGS5 PEGS5入力のRHOを
  これに揃えるよう明記する。
- **自己吸収による観測量Bの非情報化**: 上記「事前に予測しておく物理的な
  非対称性」参照。これは差異要因ではなく、この検証設計自体の限界として
  記録する。

## 判定の扱い

- 上記基準に対して不合格が出ても基準そのものを緩めない。原因調査を別項目
  として記録し、未解決のまま「条件付き合格」として残すことを許容する。
- 観測量Bが上記の予測通り非情報的だった場合、それ自体を「予測が的中した」
  という検証結果として記録し、判定は観測量Aのみで行う。
